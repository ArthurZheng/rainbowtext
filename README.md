# rainbowtext
A mini Flask App to demonstrate auto deploy to AWS-ECS Fargate with Docker-Compose

Project structure.

- nginx
    | - check.sh
    | - dockerfile
    | - nginx.conf
    | - nginx.dev.conf
- scripts
    | - .env (ignored)
    | - config_ecr.py
    | - del_ecr.py
    | - deploy.sh
    | - login_ecr.sh
    | - check.sh
    | - check.sh
    | - check.sh

- templates

- docker-compose.yml
- dockerfile
- ecs-params.yml
- instructions.txt
- uwsgi.ini
